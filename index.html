Terraform Introduction

Terraform is an Infrastructure as Code (IaC) tool by HashiCorp used to provision and manage cloud resources automatically.

terraform init
terraform plan
terraform apply
terraform destroy

Workflow Diagram:

Write Code → Init → Plan → Apply → Infrastructure Deployment

Installation

# Windows (Chocolatey)
choco install terraform

# Verify
terraform version

Providers

terraform {
  required_providers {
    azurerm = {
      source  = "hashicorp/azurerm"
      version = "~>3.0"
    }
  }
}

provider "azurerm" {
  features {}
}

Resources

resource "azurerm_resource_group" "rg" {
  name     = "dev-rg"
  location = "Central India"
}

resource "azurerm_virtual_network" "vnet" {
  name                = "dev-vnet"
  address_space       = ["10.0.0.0/16"]
  location            = azurerm_resource_group.rg.location
  resource_group_name = azurerm_resource_group.rg.name
}

Variables

variable "location" {
  type    = string
  default = "Central India"
}

variable "environment" {
  type = string
}

Outputs

output "resource_group_name" {
  value = azurerm_resource_group.rg.name
}

State Management

terraform state list
terraform state show azurerm_resource_group.rg

Remote Backend

terraform {
  backend "azurerm" {
    resource_group_name  = "tfstate-rg"
    storage_account_name = "tfstateprod123"
    container_name       = "tfstate"
    key                  = "prod.terraform.tfstate"
  }
}

Modules

module "network" {
  source = "./modules/network"
  location = var.location
}

CI/CD Integration

Git Push → GitHub Actions → Terraform Init → Plan → Approval → Apply
